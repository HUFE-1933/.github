# 湖南财政经济学院
## HUNAN UNIVERSITY OF FINANCE AND ECONOMICS

欢迎访问湖南财政经济学院 GitHub 组织主页。

本组织用于学校相关教学实践、科研创新、项目开发与技术成果的管理、展示和协作，服务于教育教学、学术研究、实践育人和校园信息化建设。

## 学校简介

湖南财政经济学院是湖南省属公办普通本科院校，由湖南省人民政府举办，湖南省教育厅主管。学校前身可追溯至 1933 年创办的厚生会计讲习所，2010 年经教育部批准升格为本科院校并定名为湖南财政经济学院。

学校以经济学、管理学为主体，突出财经特色，统筹推进相关学科协调发展，持续服务区域经济社会发展和高等教育现代化建设。
